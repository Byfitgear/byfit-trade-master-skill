# BYFIT 客户评分框架（7维120分制）

## 概述

本框架将客户评估从传统5维升级为7维，新增**社交媒体活跃度**和**采购行为匹配**两个维度，使评分更加精准地反映客户的真实价值和成交可能性。

---

## 七维评分体系

### 维度1：采购意向（Purchase Intent）— 0-25分

| 分值 | 等级 | 判定标准 |
|------|------|----------|
| 21-25 | 极强 | 明确采购计划+具体规格+时间表+已有预算 |
| 16-20 | 强 | 有采购需求+基本规格+2-3月内需要 |
| 11-15 | 中 | 有兴趣+模糊需求+时间表不确定 |
| 6-10 | 弱 | 简单询价+无具体需求+比价性质 |
| 0-5 | 极弱 | 通用询盘+群发性质+无针对性 |

**评分细则：**

| 信号 | 加分 |
|------|------|
| 提供具体产品规格（厚度/尺寸/颜色/数量） | +5 |
| 提及项目时间表或开业日期 | +4 |
| 提到预算范围 | +4 |
| 询问MOQ和交期 | +3 |
| 要求寄样 | +3 |
| 询问付款方式 | +2 |
| 提到现有供应商的问题 | +3 |
| 是回复开发信（而非主动询盘） | -2 |
| 询盘内容可复制到其他供应商 | -3 |
| 只问价格无其他信息 | -2 |

**BYFIT示例：**
- "I need 500sqm 4mm black rubber tiles for a new gym opening in March, FOB Shanghai" → 25分
- "Send me your rubber flooring catalog and price list" → 6分
- "We're building a 2000sqm fitness center, need flooring and equipment quote" → 22分

### 维度2：公司实力（Company Quality）— 0-20分

| 分值 | 等级 | 判定标准 |
|------|------|----------|
| 17-20 | 极强 | 行业知名/上市公司/年营收>$50M |
| 13-16 | 强 | 区域知名/中大型企业/年营收$10-50M |
| 9-12 | 中 | 稳定经营/中小企业/年营收$1-10M |
| 5-8 | 弱 | 小型企业/经营3年以上/年营收<$1M |
| 0-4 | 极弱 | 刚成立/信息极少/无法验证 |

**评分细则：**

| 信号 | 加分 |
|------|------|
| 行业排名Top 50（本国） | +5 |
| 员工数>100 | +3 |
| 经营年限>10年 | +3 |
| 有正规公司网站（非模板） | +2 |
| LinkedIn公司页面完整 | +2 |
| 有第三方信用评级 | +2 |
| 上市公司（公开财务可查） | +5 |
| 有进出口记录 | +3 |
| 无公司网站或网站极简 | -3 |
| 注册不满1年 | -3 |
| 无法在公司注册系统查到 | -5 |
| 邮箱为个人邮箱（gmail/hotmail） | -2 |

**BYFIT示例：**
- Planet Fitness（NYSE: PLNT，2400+门店）→ 20分
- Local gym with 3 locations, 10 years in business → 12分
- Newly opened studio, no website → 4分

### 维度3：采购潜力（Volume Potential）— 0-20分

| 分值 | 等级 | 判定标准 |
|------|------|----------|
| 17-20 | 极高 | 年采购额>$500K或持续大批量需求 |
| 13-16 | 高 | 年采购额$100-500K或定期中量需求 |
| 9-12 | 中 | 年采购额$20-100K或不定期中小量需求 |
| 5-8 | 低 | 年采购额$5-20K或单次小量需求 |
| 0-4 | 极低 | 年采购额<$5K或单次极小量 |

**评分细则：**

| 信号 | 加分 |
|------|------|
| 海关数据显示年进口额>$200K | +5 |
| 门店数>50（连锁） | +5 |
| 有新店扩张计划 | +4 |
| 询盘中提到年采购量 | +3 |
| 多产品线需求（flooring+equipment） | +3 |
| 项目型需求（新建/翻新多个场所） | +4 |
| 只需样品/小批量测试 | -2 |
| 单次采购无复购预期 | -3 |
| 明确表示"just testing" | -4 |

**BYFIT示例：**
- 200+门店连锁，每年新开30+店，每店需200sqm flooring + 全套设备 → 20分
- 单店采购500sqm flooring → 8分
- 5个工作室的Pilates连锁 → 13分

### 维度4：市场匹配（Market Fit）— 0-20分

| 分值 | 等级 | 判定标准 |
|------|------|----------|
| 17-20 | 完美 | 产品完全匹配客户需求+市场+价格带 |
| 13-16 | 高度匹配 | 核心产品匹配+市场合适+价格基本匹配 |
| 9-12 | 中度匹配 | 部分产品匹配+市场可行+价格需调整 |
| 5-8 | 低度匹配 | 产品勉强匹配+市场有壁垒+价格差距大 |
| 0-4 | 不匹配 | 产品不匹配或市场不可行 |

**评分细则：**

| 信号 | 加分 |
|------|------|
| 客户需求与BYFIT核心产品线完全一致 | +5 |
| 客户所在市场是BYFIT目标市场 | +4 |
| 客户价格带与BYFIT定位匹配 | +3 |
| 无重大合规/认证壁垒 | +3 |
| 客户有相关产品采购经验 | +3 |
| 需要BYFIT不具备的认证 | -4 |
| 市场有高关税壁垒 | -3 |
| 价格带差距>30% | -4 |
| 需要BYFIT不生产的产品类型 | -3 |

**BYFIT示例：**
- 美国商业健身连锁，采购rubber flooring + gym equipment，无特殊认证需求 → 18分
- 德国客户需要特定防火等级B1（BYFIT产品未认证）→ 10分
- 印度客户要求最低价rubber tiles（低于BYFIT成本）→ 4分

### 维度5：沟通质量（Communication Quality）— 0-15分

| 分值 | 等级 | 判定标准 |
|------|------|----------|
| 13-15 | 优秀 | 沟通频繁+内容具体+响应及时+主动提供信息 |
| 10-12 | 良好 | 沟通正常+内容较具体+响应较及时 |
| 6-9 | 一般 | 沟通不规律+内容简短+响应延迟 |
| 3-5 | 较差 | 偶尔回复+内容模糊+响应很慢 |
| 0-2 | 极差 | 几乎不回复或只有自动回复 |

**评分细则：**

| 信号 | 加分 |
|------|------|
| 24小时内回复邮件 | +3 |
| 邮件内容详细，回答所有问题 | +3 |
| 主动提供额外信息（公司介绍、项目详情） | +3 |
| 使用公司邮箱（非个人邮箱） | +2 |
| 沟通语气专业友好 | +2 |
| 答非所问或只回答部分问题 | -2 |
| 需要3+天才能回复 | -2 |
| 只用WhatsApp/WeChat不愿发邮件 | -1 |
| 英语/沟通语言能力差导致误解 | -2 |

### 维度6：社交媒体活跃度（Social Media Activity）— 0-10分 【新增】

| 分值 | 等级 | 判定标准 |
|------|------|----------|
| 9-10 | 极活跃 | 多平台活跃+频繁更新+高互动+招聘信号+行业影响力 |
| 7-8 | 活跃 | 2-3个平台活跃+定期更新+有互动 |
| 4-6 | 中等 | 1-2个平台有存在+偶尔更新+低互动 |
| 1-3 | 低 | 有账号但不活跃+很少更新 |
| 0 | 无 | 找不到社交媒体存在 |

**评分细则：**

| 信号 | 加分 |
|------|------|
| LinkedIn公司页面：500+关注者，月更新4+次 | +2 |
| 决策人LinkedIn活跃（周更+互动） | +2 |
| 正在招聘相关岗位（采购/运营/扩张） | +2 |
| Instagram/Facebook：定期发帖+高互动 | +1 |
| 近期发布新产品/新店/扩张计划 | +2 |
| YouTube有品牌频道且定期更新 | +1 |
| 社交媒体完全空白 | -1 |
| 最后更新超过1年 | -1 |
| 负面评价或投诉集中 | -2 |

**BYFIT示例：**
- 健身连锁，LinkedIn 5000+关注者，月更8次，正在招聘3个新店经理 → 10分
- 小型工作室，有Instagram但不常更新（月1-2次），LinkedIn无公司页面 → 3分
- 设备经销商，LinkedIn活跃，正在招聘"sourcing specialist" → 8分

**社交媒体检查清单：**
```
□ LinkedIn公司页面：关注者数、更新频率、帖子内容
□ LinkedIn决策人：职位、活跃度、发帖内容
□ 招聘信号：是否在招采购/运营/新店岗位
□ Facebook：公司页面活跃度、评价、群组参与
□ Instagram：发帖频率、hashtag使用、互动率
□ YouTube：频道存在、视频内容、评论
□ Twitter/X：是否存在、发帖内容
□ Google Reviews：评价数量和评分
```

### 维度7：采购行为匹配（Purchasing Behavior Match）— 0-10分 【新增】

| 分值 | 等级 | 判定标准 |
|------|------|----------|
| 9-10 | 极高匹配 | 有同品类进口记录+采购周期与BYFIT匹配+供应商切换信号 |
| 7-8 | 高匹配 | 有相关品类进口记录+采购周期基本匹配 |
| 4-6 | 中匹配 | 有进口记录但品类不完全匹配/周期不确定 |
| 1-3 | 低匹配 | 无进口记录或品类差距大 |
| 0 | 无匹配 | 无任何进口记录或明确不进口该品类 |

**评分细则：**

| 信号 | 加分 |
|------|------|
| 海关数据显示进口过rubber flooring/gym equipment | +3 |
| 采购周期与BYFIT交期匹配（提前2-3月下单） | +2 |
| 现有供应商出现问题的信号（投诉/更换/延迟） | +2 |
| 采购量级与BYFIT产能匹配 | +2 |
| 采购国家为中国（说明接受中国供应商） | +1 |
| 无进口记录（可能走贸易商/无经验） | -2 |
| 只从本国/非中国地区采购 | -2 |
| 采购频次极低（<1次/年） | -1 |

**BYFIT示例：**
- 海关数据：2024年从山东某厂进口3批rubber flooring，总$180K，最近一批3月 → 9分
- 海关数据：进口过fitness equipment但无flooring记录 → 6分
- 无海关数据但有Alibaba询盘记录 → 4分
- 明确表示"we only buy from Europe" → 0分

**采购行为检查清单：**
```
□ 海关数据：是否有进口记录
□ 进口品类：是否包含BYFIT相关品类
□ 供应商分析：现有供应商是谁、关系多长
□ 采购周期：下单频率和提前期
□ 采购量级：单次/年度采购量
□ 供应商切换信号：是否在寻找新供应商
□ 付款习惯：T/T、L/C、O/A？
□ 贸易条款偏好：FOB、CIF、DDP？
```

## 等级划分

| 等级 | 分数范围 | 客户特征 | 资源投入 | 目标 |
|------|----------|----------|----------|------|
| **Tier 1** | 90-120 | 高价值+高意向+大潜力 | 全力投入 | 快速成交+深度绑定 |
| **Tier 2** | 60-89 | 有价值+中等意向+有潜力 | 重点投入 | 加速转化+培育升级 |
| **Tier 3** | 30-59 | 有限价值+低意向 | 适度投入 | 长期培育+等待升级信号 |
| **Tier 4** | 0-29 | 低价值/风险客户 | 最少投入 | 自动化维护+监控 |

## 各等级跟进节奏

### Tier 1（90-120分）— 全力推进

```
Day 1:    收到询盘 → 30分钟内回复 → 完成7维评分 → 启动10步Email Group
Day 2:    发送Welcome Email + Value Proposition
Day 3:    发送Product Deep-Dive + 样品安排
Day 4:    发送Social Proof + 案例分享
Day 5:    电话/视频会议邀请
Day 7:    发送Comparison Email
Day 8:    发送Objection Pre-emption
Day 10:   发送Urgency Email
Day 12:   发送Risk Reversal Email

Day 16:   发送Close Email
Day 17:   如未回复，电话跟进

Day 21:   如仍未成交，切换为培育模式
---
后续：每周触达1次，月度深度沟通1次
目标：30天内成交
```

### Tier 2（60-89分）— 重点培育

```
Day 1:    收到询盘 → 2小时内回复 → 完成7维评分 → 启动5-6步Email Group

---

目标：45天内成交或升级
```

### Tier 3（30-59分）— 长期培育

```

```

### Tier 4（0-29分）— 自动化维护

```

---

目标：监控升级信号，自动提升等级
无人工投入
```

## 等级升降触发条件

### 升级触发（自动升级，无需等待评估周期）

| 从 → 到 | 触发条件 |
|---------|----------|
| Tier 4 → Tier 3 | 客户主动回复/询问具体产品/网站访问增加 |
| Tier 3 → Tier 2 | 提供具体规格/要求报价/提到采购时间表/社交媒体发布扩张计划 |
| Tier 2 → Tier 1 | 确认采购量/要求样品/安排验厂/讨论合同条款/提出拜访 |
| 跳级（Tier 3→1） | 大额询盘/上市公司招标/知名品牌直接合作意向 |

### 降级触发（评估后降级）

| 从 → 到 | 触发条件 |
|---------|----------|
| Tier 1 → Tier 2 | 连续3次未回复/明确表示暂不采购/决策人离职 |
| Tier 2 → Tier 3 | 连续5次未回复/超过60天无互动/明确拒绝报价 |
| Tier 3 → Tier 4 | 连续8次未回复/超过90天无互动/邮箱退信 |
| 直接移除 | 公司倒闭/欺诈嫌疑/明确表示不再需要 |

### 降级冷却期

- 从Tier 1降至Tier 2后，30天内不再降级（给予恢复机会）
- 从Tier 2降至Tier 3后，45天内不再降级
- 降级后首次触达使用更轻量的方式

## 快速评分速查表

### 一分钟快速评分

在时间有限时，用以下简化方法快速估算：

```
快速判断（每项1-5分）：

1. 客户说了具体要什么？         [1=没说  5=非常具体]  ×5 = ___
2. 客户公司有多大？             [1=很小  5=很大]      ×4 = ___
3. 能买多少？                   [1=很少  5=很多]      ×4 = ___
4. 和我们产品匹配吗？           [1=不匹配 5=完美]    ×4 = ___
5. 沟通顺畅吗？                 [1=很差  5=很好]      ×3 = ___
6. 社交媒体活跃吗？             [1=没有  5=很活跃]    ×2 = ___
7. 买过类似产品吗？             [1=没买过 5=常买]    ×2 = ___

总分 = ___ /120
```

### 常见客户类型快速评分参考

| 客户类型 | 预估分数 | 等级 |
|----------|----------|------|
| 大型健身连锁询盘（有规格有量有时间） | 100-115 | Tier 1 |
| 中型设备经销商（有经验有量） | 80-95 | Tier 1-2 |
| 小型健身房业主（单店采购） | 45-65 | Tier 2-3 |
| Alibaba通用询盘（群发性质） | 15-35 | Tier 3-4 |
| 品牌商（OEM/ODM需求） | 75-100 | Tier 1-2 |
| 政府项目采购 | 65-90 | Tier 1-2 |
| 创业者（新开健身房） | 30-55 | Tier 3 |
| 展会名片（无后续互动） | 10-25 | Tier 4 |

## 评分常见错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
