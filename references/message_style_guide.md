# 多渠道消息风格指南 (Multi-Channel Message Style Guide)

> 核心原则：不同渠道有不同的沟通语法。邮件像商务信函，WeChat像面对面聊天，WhatsApp像发短信。用错风格 = 破坏专业形象 = 失去信任。

## 渠道概览与选择

### 三大渠道对比

| 维度 | Email | WeChat | WhatsApp |
|------|-------|--------|----------|
| 正式程度 | 高 | 中 | 中低 |
| 典型长度 | 150-300词 | 80-150字 | 60-120词 |
| 响应期望 | 4-24小时 | 1-4小时 | 15-60分钟 |
| 适合内容 | 报价、方案、正式通知 | 日常沟通、关系维护 | 快速确认、提醒、简短跟进 |
| 文件附件 | 支持（但别太大） | 支持 | 支持 |
| 语气 | 专业、结构化 | 温暖、对话感 | 直接、友好 |
| 格式 | 完整格式（标题+问候+正文+CTA+签名） | 灵活 | 简洁 |
| 文化偏好 | 全球通用 | 中国+华人客户 | 欧洲、中东、拉美、东南亚 |

### 渠道选择决策树

```
你要发什么消息？
│
├─ 正式报价/方案/合同 ──→ Email（必须）
│   └─ 报价后24小时无回复 ──→ WhatsApp提醒"请查收邮件"
│
├─ 日常沟通/问题讨论 ──→ 看客户偏好
│   ├─ 客户习惯用WeChat ──→ WeChat
│   ├─ 客户习惯用WhatsApp ──→ WhatsApp
│   └─ 不确定 ──→ Email
│
├─ 快速确认/提醒 ──→ WhatsApp 或 WeChat
│
├─ 关系维护/节日问候 ──→ 看客户文化
│   ├─ 中国/华人客户 ──→ WeChat
│   ├─ 中东/欧洲/拉美 ──→ WhatsApp
│   └─ 通用 ──→ Email + LinkedIn
│
└─ 紧急情况 ──→ 电话 > WhatsApp/WeChat > Email
```

## Email 风格指南

### 基本规范

| 规范 | 要求 |
|------|------|
| 字数 | 150-300词（正文） |
| 段落 | 3-5段，每段2-4句 |
| 主题行 | 5-10词，包含关键信息 |
| 问候语 | Hi [First Name] 或 Dear [First Name] |
| 结尾 | Best regards / Best / Cheers + 签名档 |
| CTA | 每封邮件一个明确的行动号召 |
| 附件 | 控制在5MB以内，用PDF格式 |
| 回复时间 | 目标4小时内，最长24小时 |

### ❌ 自带AI味的句式（看到立刻改）

下面这些句式看起来"很专业"但实际上每个客户都见过几百遍了，看到就删：

| AI味句式 | 为什么不行 | 改成 |
|----------|-----------|------|
| "I hope this email finds you well." | 99%的AI销售邮件以它开头 | 直接说事或直接问问题 |
| "I am writing to introduce our company..." | 太正式，像群发 | "I noticed you carry gym flooring—wanted to connect." |
| "We are a leading manufacturer of..." | 谁都说自己是leading | "We make rubber flooring in Qingdao. 20 lines, 12 certs." |
| "Please don't hesitate to reach out." | 客气过头，像客服自动回复 | "Happy to send specs if you need them." |
| "Our commitment to quality..." | 空话，没信息量 | "EN14904 certified. Batch QC on every shipment." |
| "At BYFIT, we pride ourselves on..." | 可以删掉仍不影响内容 | 直接删，从"we"开始就好 |
| "We understand that price is an important factor." | AI废话 | "Price matters. Here's how ours stacks up." |
| "If you have any questions, please feel free to ask." | 太长了 | "Questions? Just ask." 或直接不问 |
| "We look forward to building a successful partnership." | 太早、太假、像模板 | 不急着谈partnership，先谈需求 |
| "Thank you in advance for your kind consideration." | 舔过头 | "Appreciate your time." |
| "I would like to take this opportunity to..." | 多余的装饰语 | 删掉整句话 |
| "Furthermore / Moreover / Additionally" | AI信号词 | 直接说下一句，不用转 |

### 邮件结构

```
Subject: [关键信息] — BYFIT [产品类别]

Hi [First Name],

[第一段：目的/背景 — 1-2句]

[第二段：核心内容 — 2-4句]

[第三段：补充/价值 — 1-2句，可选]

[CTA：明确的下一步行动]

Best regards,
[Your Name]
[Title] | BYFIT
Gym Rubber Flooring | Gym Equipment | Pilates Equipment
WhatsApp: +86-XXX-XXXX-XXXX
Email: [email]
```

### 语气规则

| 规则 | 说明 | 示例 |
|------|------|------|
| 专业但不冷漠 | 你是专业人士，不是机器人 | "I understand your concern" > "We note your feedback" |
| 简洁但不生硬 | 每句话都有价值 | "Here's the quotation" > "Please find attached herewith the quotation for your perusal" |
| 自信但不傲慢 | 你知道自己产品好，但尊重客户 | "BYFIT offers a <1% return rate" > "BYFIT is the best supplier" |
| 友好但不卑微 | 平等对话，不是求人 | "Shall we discuss?" > "If you could please kindly consider..." |

### 邮件 Do's and Don'ts

| Do | Don't |
|----|-------|
| 用对方的名字称呼 | 用 "Dear Sir/Madam" |
| 每封邮件一个CTA | 一封邮件里问5个问题 |
| 用数字和具体数据 | 用 "competitive price" / "high quality" |
| 24小时内回复 | 3天后才回复 |
| 检查拼写和语法 | 发送前不校对 |
| 用PDF附件 | 用Word/Excel附件（不专业） |
| 主题行写明内容 | 主题行写 "Hi" 或 "Question" |
| 适当使用项目符号 | 写大段密集文字 |

## WeChat 风格指南

### 基本规范

| 规范 | 要求 |
|------|------|
| 字数 | 80-150字 |
| 语气 | 温暖、对话感、有人情味 |
| Emoji | 适度使用（1-3个/条） |
| 响应时间 | 1-4小时内 |
| 适合内容 | 日常沟通、关系维护、快速讨论 |
| 不适合 | 正式报价、合同条款、复杂技术讨论 |

### 语气规则

| 规则 | 说明 |
|------|------|
| 像面对面聊天 | 你不会面对面说"Dear Sir"，微信也不用 |
| 适度emoji | 增加温度，但不要刷屏 |
| 控制长度 | 超过150字就该考虑发邮件 |
| 及时回复 | 微信是即时通讯，6小时不回 = 不尊重 |
| 语音慎用 | 除非对方先发语音，否则用文字 |
| 不发长文 | 大段文字在手机上阅读体验差 |

### WeChat 沟通节奏

**初次添加好友后**：

```
1. 自我介绍（简洁）
   "Hi [Name], 我是BYFIT的[Your Name]，做gym flooring和
   equipment的。展会认识的，以后多交流！🤝"

2. 等对方回复（不要连发）

3. 自然过渡到业务话题
   "对了，看到你们也在做flooring，我们最近新出了EPDM彩色
   系列，有机会可以了解一下"
```

**日常沟通**：

```
✅ "David哥，报价单已经发到你邮箱了，有空看一下，有疑问随时找我"
✅ "今天上海降温了🥶，你们那边天气怎么样？"
✅ "新款pilates reformer出了，给你发个图片看看👇"

❌ "Dear Mr. David, Please find attached the quotation for your review. Should you have any questions, please do not hesitate to contact us."
❌ （连发5条60秒语音）
❌ "您好，我们BYFIT是专业的gym flooring制造商，拥有15年经验..."
```

### WeChat Do's and Don'ts

| Do | Don't |
|----|-------|
| 简短直接 | 写长篇大论 |
| 适度emoji增加温度 | emoji刷屏 |
| 发图片/视频展示产品 | 只发文字描述产品 |
| 用"哥/姐"拉近关系（中国客户） | 称呼过于正式 |
| 及时回复 | 意味着随时在线——也要有自己的节奏 |
| 业务和闲聊穿插 | 只聊业务（太功利） |
| 用文字为主 | 未经允许发语音 |

## WhatsApp 风格指南

### 基本规范

| 规范 | 要求 |
|------|------|
| 字数 | 60-120词 |
| 语气 | 直接、友好、手机优先 |
| 格式 | 短句、多用换行、适当用bold/emoji |
| 响应时间 | 15-60分钟 |
| 适合内容 | 快速确认、提醒、简短跟进、实时沟通 |
| 不适合 | 复杂报价、长篇方案、正式文件 |

### 语气规则

| 规则 | 说明 |
|------|------|
| 手机友好 | 短句、多换行——对方在手机上看 |
| 直接 | 说重点，不要铺垫 |
| 友好 | WhatsApp是私人空间，语气要像朋友 |
| 快速 | 这不是邮件，不要隔天回复 |
| 不过度 | 不要一条接一条轰炸 |

### BYFIT WhatsApp 示例

**首次联系（从邮件转WhatsApp）**：

```
"Hi David, this is [Your Name] from BYFIT — we've been
exchanging emails about rubber flooring. Thought WhatsApp
might be easier for quick questions. 👋"
```

**报价提醒**：

```
"Hi David, just sent the updated quotation to your email.
Key change: EPDM option now at $16.80/sqm (was $17.50).
Let me know if you need anything else! 📧"
```

**生产进度更新**：

```
"Production update for your order #12345:
✅ Raw materials: Ready
✅ Production: Started today
⏳ Estimated completion: June 20
📦 Shipping: June 22

I'll send photos when the tiles come off the line! 📸"
```

**紧急催单**：

```
"Hi David, just a heads up — our June production slots are
almost full. If you want to secure this order, we'd need
confirmation by Friday. Let me know! ⏰"
```

**周末轻松聊天**：

```
"Happy Friday David! Any plans for the weekend? 😊

By the way, I'll send those sample photos on Monday."
```

### WhatsApp Do's and Don'ts

| Do | Don't |
|----|-------|
| 简短消息 | 写长段落 |
| 用✅⏳📦等状态emoji | 用过于花哨的emoji |
| 发图片/短视频展示产品 | 发大文件 |
| 快速回复 | 已读不回 |
| 发语音（对方先发的话） | 不问就发长语音 |
| 尊重对方时区 | 对方凌晨发消息 |
| 用WhatsApp做快速确认 | 用WhatsApp发正式合同 |

## 跨渠道消息一致性

### 原则：消息一致，风格适配

**同一件事在不同渠道上说，核心信息必须一致，但表达方式根据渠道调整。**

**场景：通知客户样品已发货**

| 渠道 | 消息 |
|------|------|
| Email | "Dear David, I'm pleased to inform you that your sample shipment has been dispatched via DHL. Tracking number: 123456789. Estimated delivery: June 15. The package contains: 5 rubber tile samples (mixed specs/colors) + product spec sheets + installation guide. Please let me know when you receive them." |
| WhatsApp | "Hi David, samples shipped! 📦 DHL tracking: 123456789. ETA: June 15. 5 tiles + spec sheets inside. Let me know when they arrive!" |
| WeChat | "David，样品已发📦 DHL单号123456789，大概6月15号到。里面5块tile+资料，到了跟我说👍" |

### 跨渠道消息流转规则

1. **邮件 → WhatsApp**：发送重要邮件后，WhatsApp简短提醒对方查看
2. **WhatsApp → 邮件**：WhatsApp讨论的关键决定，邮件确认（书面留痕）
3. **WeChat → 邮件**：同上
4. **电话 → 邮件**：电话讨论后，邮件总结确认（"Per our conversation..."）

### 消息冲突处理

**如果邮件和WhatsApp/WeChat传达了不同信息**：
1. 以邮件为准（书面记录）
2. 立即在所有渠道上发出更正
3. 承认错误，不找借口

**示例**：

```
邮件报了$14.80，WhatsApp不小心说了$13.80：

WhatsApp更正：
"David, quick correction — I made a typo earlier. The correct
price is $14.80/sqm (not $13.80). Apologies for the confusion!"

邮件确认：
"Hi David, just to confirm, the correct FOB price for rubber
tiles is $14.80/sqm as stated in the original quotation.
Apologies for any confusion from my WhatsApp message."
```

## 核心金句

> "邮件是你的商务西装，WhatsApp是你的休闲装，WeChat是你的家居服。场合不对，形象崩塌。"

> "同一个消息，在邮件里叫'正式通知'，在WhatsApp里叫'快速提醒'，在WeChat里叫'跟你说一声'。内容一样，温度不同。"

> "不要在WhatsApp里写邮件，也不要在邮件里发微信——每个渠道都有自己的语法。"

> "跨渠道一致性不是'复制粘贴'，是'核心信息一致 + 表达方式适配'。"
